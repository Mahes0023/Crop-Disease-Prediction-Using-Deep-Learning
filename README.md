# 🌿 AI Crop Monitoring System

An end-to-end AI-powered crop disease detection system that analyzes plant leaf images and provides actionable insights including disease prediction, soil condition analysis, pest risk, and treatment recommendations.

⸻

# 🚀 Features
	•	📸 Upload crop leaf images
	•	🤖 AI-based disease prediction using TensorFlow
	•	🌱 Soil condition insights
	•	🐛 Pest risk analysis
	•	💊 Treatment recommendations
	•	⚡ Real-time prediction via Flask API
	•	🎨 Modern, responsive UI with image preview and loading indicators

⸻

# 🧠 How It Works
	1.	User uploads a crop image via the web interface
	2.	Image is sent to the Flask backend
	3.	Preprocessing is applied (resize, normalization)
	4.	TensorFlow model predicts disease
	5.	System generates:
	•	Disease classification
	•	Confidence score
	•	Soil condition insight
	•	Pest risk level
	•	Recommended solution
	6.	Results are displayed in a clean UI

⸻

	•	Fixing TensorFlow model loading errors (quantization_config)
	•	Ensuring consistent API response handling in frontend

# 📌 Conclusion

The AI Crop Monitoring System demonstrates how machine learning can be effectively integrated into real-world agricultural applications to provide intelligent, data-driven insights. By combining a trained deep learning model with a responsive web interface, this project delivers an end-to-end solution that is both practical and scalable.

This project not only highlights the capability of AI in early disease detection but also emphasizes the importance of seamless backend–frontend integration, efficient API design, and user-centric UI development. Overcoming challenges such as model compatibility issues and cross-origin communication further strengthened the robustness of the system.

Overall, this system serves as a strong foundation for building advanced agricultural intelligence platforms and reflects the potential of technology in enhancing productivity, sustainability, and decision-making in farming.
